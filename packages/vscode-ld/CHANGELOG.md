# 更新日志

## [0.2.0] - 2024-01-XX

### 新增
- 🎯 **Prettier集成**：使用Prettier进行更精确的代码格式化
- 🔍 **错误检测**：实时诊断LD文件中的语法错误和潜在问题
- 📝 **代码片段**：丰富的代码片段支持，快速生成常用代码结构
- 🎨 **图标支持**：为`.ld`文件添加专用图标
- ⚡ **性能优化**：优化诊断和格式化性能

### 改进
- 格式化器支持分段格式化（template、script、style分别处理）
- 诊断提供者检测未闭合标签、CSS错误等
- 代码片段包含Vue3、React和LD API的常用模式
- 更好的错误提示和文档

### 功能
- Prettier格式化支持（可配置）
- 实时错误检测和诊断
- 代码片段自动补全
- 文件图标识别
